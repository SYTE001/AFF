# 🛒 AFF - Smart Bridge Page
> Solusi pengganti Keranjang Kuning TikTok untuk pejuang Affiliate Shopee.

[![Status](https://img.shields.io/badge/Status-Active-brightgreen.svg)]()
[![Platform](https://img.shields.io/badge/Platform-Mobile--First-blue.svg)]()

### 💡 Latar Belakang
Karena keterbatasan fitur Keranjang Kuning (syarat 600 followers), **AFF** hadir sebagai "jembatan" profesional. Web ini menampung semua link produk rekomendasi dari konten TikTok/IG kamu dalam satu halaman yang ringan dan cepat.

---

### ✨ Fitur Utama
* **Mobile-Optimized:** Tampilan super responsif, nyaman dibuka dari in-app browser TikTok/Instagram.
* **Clean UI:** Fokus pada produk, tanpa gangguan layout yang rumit.
* **Direct-to-App:** Link yang diklik otomatis mengarah ke aplikasi Shopee.
* **Ultra Lightweight:** Load halaman sangat cepat agar calon pembeli tidak kabur.

### 🛠️ Tech Stack
* **HTML5 & CSS3** (Custom Styling)
* **Modern Icons** (Lucide/FontAwesome)
* **Deployment:** GitHub Pages / Vercel

---

### 🚀 Cara Pakai (Setup)
1.  **Clone Repo:** `git clone https://github.com/SYTE001/AFF.git`
2.  **Update Link:** Buka file utama (misal `index.html`), cari bagian link, dan ganti dengan link affiliate Shopee punyamu.
3.  **Deploy:** Push ke GitHub Pages dan pasang link web-nya di Bio TikTok/IG kamu.
